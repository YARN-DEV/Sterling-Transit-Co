# Sterling Transit Co Frontend

This is the Next.js frontend for Sterling Transit Co, using React and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```
npm install
   ```
2. Run the development server:
   ```
npm run dev
   ```

## Project Structure
- `pages/` - Next.js pages
- `components/` - React components
- `styles.css` - Tailwind CSS entry
